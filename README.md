# 🚴 BikeTrain

BikeTrain 是一個使用 Flask 開發的自行車訓練平台。

使用者可以：

- 註冊帳號
- 登入系統
- 紀錄騎乘資料
- 查看訓練統計
- 計算 FTP 功率區間
- 查看 W/kg
- 獲得訓練建議
- 使用課表產生器

---

## 技術架構

### Backend

- Flask
- Flask-Login
- Flask-WTF
- SQLAlchemy

### Database

- MySQL

### Frontend

- Bootstrap 5
- Chart.js
